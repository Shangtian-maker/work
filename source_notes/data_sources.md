# week1_public_samples 数据来源与获取说明


## 一、文件说明
根据目标公司名单生成公司清单。

输出文件：
- `week1_public_samples.csv`
- `week1_public_samples_data_sources.md`

## 二、字段说明

| 字段 | 说明 |
|---|---|
| sample_id | 样本编号 |
| company_name | 公司名称 |
| stock_code | 股票代码 |
| exchange | 证券交易所 |
| board | 上市板块 |
| listing_date | 上市日期，部分公司需根据上市公告书继续核验 |
| ipo_year | IPO年份，优先取上市日期年份；如缺失则取招股书披露年份 |
| source_platform | 数据来源平台 |
| source_page_url | 来源页面或PDF链接 |
| prospectus_title | 招股说明书标题 |
| prospectus_url | 招股说明书PDF链接 |
| prospectus_version | 文件版本，如招股说明书、注册稿、申报稿 |
| prospectus_date | 招股说明书披露日期 |
| download_status | ready表示已有PDF链接，pending表示需继续定位PDF |
| parse_status | PDF解析状态，默认pending |
| locate_status | 是否已定位来源页面 |
| extract_status | 当前字段提取状态 |
| review_status | 人工复核状态 |
| notes | 备注和待复核事项 |

## 三、数据来源

1. 上海证券交易所公告页
适用公司：
- 天和磁材
- 友升股份
- 赛分科技
- 影石创新

适用板块：
- 主板
- 科创板

检索关键词：
- 公司简称 + 招股说明书
- 公司简称 + 首次公开发行股票并在主板上市招股说明书
- 公司简称 + 首次公开发行股票并在科创板上市招股说明书

2. 深圳证券交易所公告页
适用公司：
- 黄山谷捷
- 云汉芯城

适用板块：
- 创业板

检索关键词：
- 公司简称 + 招股说明书
- 公司简称 + 首次公开发行股票并在创业板上市招股说明书

3. 北京证券交易所信息披露页
适用公司：
- 三协电机
- 大鹏工业

适用板块：
- 北交所

检索关键词：
- 公司简称 + 招股说明书
- 公司简称 + 公开发行股票并在北京证券交易所上市招股说明书

4. 巨潮资讯网公告
5. 东方财富公告页
6. 新浪财经公告页

## 四、获取方法

### 1. 交易所官网

按公司名称或股票代码搜索：

- 上交所：主板、科创板公司
- 深交所：创业板公司
- 北交所：北交所公司

关键词示例：

- `天和磁材 招股说明书 上交所`
- `云汉芯城 招股说明书 深交所`
- `三协电机 招股说明书 北交所`

### 2. 巨潮资讯网

适合检索 A 股正式公告 PDF。

关键词示例：

- `公司简称 + 首次公开发行股票 + 招股说明书`
- `股票代码 + 招股说明书`

### 3. 东方财富 / 新浪财经

用于补充公告页面、上市公告书、招股说明书提示性公告等信息。
