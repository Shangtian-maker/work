# 招股说明书下载方法说明

1. 下载目标
下载以下八家目标公司招股说明书 PDF：

三联锻造
友升股份
黄山谷捷
云汉芯城
赛分科技
影石创新
三协电机
星图测控

确保 PDF 文件完整、可读取，便于后续解析和信息抽取。

2. 下载渠道
主要渠道：

上交所、深交所、北交所官网披露栏目
国家企业信用信息公示系统（部分信息可补充）

备用渠道：

巨潮资讯网公告
东方财富公告页
新浪财经公告页

3. 文件命名规则

文件名建议统一规范：
公司简称_招股说明书_年份.pdf

4. PDF抓取规则
在下载目标公司招股说明书时，需要遵循以下抓取规则，确保获取的 PDF 为正式招股书而非非正式文件：

4.1 关键词过滤
必须包含关键词：招股说明书
排除关键词：申报稿、注册稿、提示性公告 等非正式文件标题词
下载脚本在抓取页面时，应优先匹配包含必须关键词且不含排除关键词的链接。

4.2 文件大小筛选
正式招股说明书文件通常体积较大（例如 >1MB），提示性公告或简要文件通常体积较小。
载脚本在获取 PDF 后，应检查文件大小，过滤过小的文件以避免抓取非正式文件。

4.3 PDF页数筛选
正式招股说明书一般页数较多，通常在 200 页以上。
下载脚本可在下载或解析 PDF 时统计页数，筛选页数不足的文件。

4.4 URL匹配与页面定位
使用公司名称和“招股说明书”作为关键匹配标识，抓取对应下载链接。
对不同交易所网站，可根据页面结构定位链接，例如：
上交所：<a> 标签内包含公司简称 + “招股说明书”
深交所：<div class="file-list"> 下的 <a> 标签

4.5 下载规则与异常处理
检查文件类型，确保下载的是 PDF。
文件命名按照规则生成：公司简称_招股说明书_年份.pdf
对下载失败或不符合规则的 PDF，记录在日志 logs/download_log.csv。

4.6 抓取频率与延迟
为避免网站访问限制，每次请求可随机延迟 1~3 秒。
建议批量下载分批执行，以降低 IP 封禁风险。

5. 下载步骤

1.准备公司清单
  使用八家目标公司列表，整理 CSV 或直接使用脚本内列表。

2.执行下载脚本
  脚本存放路径：code/03_download_pdfs/
  脚本功能：
     根据公司名称匹配下载渠道。
     批量下载 PDF。
     保存到指定目录。

3.记录日志
  日志文件：logs/download_log.csv
  日志内容应包括：
     公司名称
     下载 URL
     PDF 保存路径
     下载状态（成功/失败）
     错误信息（如有）

6. 注意事项
访问限制：
有些网站可能需要登录或验证码，需提前处理。

网络稳定性：
建议断点续传或分批下载，避免网络中断导致文件不完整。

文件完整性：
下载完成后检查 PDF 可打开，文件大小合理。

失败样本处理：
下载失败或文件损坏的样本需记录在 logs/download_log.csv，便于后续手动补充
